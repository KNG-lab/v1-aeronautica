# Imágenes locales del hero

Guardá aquí la imagen cinematográfica del hero con este nombre EXACTO:

    v1-hero-cinematic.webp

Es la foto de la tripulante de cabina con uniforme V1 frente a la turbina
encendida, pista mojada y atardecer (sin texto ni botones incrustados).

- Formato recomendado: **WebP** o AVIF, ~1920×1080 o mayor, optimizada (< 400 KB).
- La composición debe dejar **espacio oscuro a la izquierda** para el copy, con la
  turbina al centro-derecha y la persona a la derecha.
- El componente del hero (`src/components/public/Hero.tsx`) ya la referencia en
  `/images/v1-hero-cinematic.webp`. Mientras el archivo no exista, se muestra un
  fallback oscuro premium para que el layout no se rompa.
